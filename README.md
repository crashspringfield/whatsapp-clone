# About
(in progress) Demo WhatsApp clone written in TypeScript to practice Postgres & GraphQL integration w/ Node.js && React.

# Gettings Started
* `sudo -u postgres -i`
* `psql`
* `CREATE DATABASE Whatsapp`

## Node Server
* `cd node-server`
* `yarn install`
* `yarn dev`

## React Client
* `cd react-client`
* `yarn install`
* `yarn start`
